# 💣 BOMB DEFUSAL — 1:1 協力型 爆弾解除ゲーム

2人のプレイヤーが WebSocket（TCP）でリアルタイム通信しながら協力して爆弾を解除するゲーム。
プレイヤーA（爆弾担当）は爆弾画面のみ、プレイヤーB（マニュアル担当）は解除マニュアルのみが見える非対称協力ゲーム。

## 通信方式

- **プロトコル:** WebSocket（TCP ベース）
- **通信形態:** 1:1（A ↔ サーバー ↔ B）
- **メッセージ形式:** JSON

## 起動方法

```bash
# 依存パッケージのインストール
npm install

# サーバー起動
npm start
# → http://localhost:3000 にアクセス
```

2つのブラウザタブ（または2台の端末）で開く:
1. 1人目: 「ルームを作成する」→ 表示された6桁コードを相手に伝える
2. 2人目: コードを入力して「参加」
3. 両者がロール（A/B）を選択するとゲーム開始

## 役割分担

| 担当 | メンバー | ファイル |
